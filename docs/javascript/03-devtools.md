При написании кода всегда будут оповещения и ошибки, это нормально. Консоль
браузера показывает информацию связанную с веб-страницей, в том числе сообщения
об ошибках и предупреждения исполнения JS-кода, запущенного в контексте
страницы.

Следующие комбинации клавишь открывают инструменты разработчика на вкладке
`Console`:

- Windows и Linux - `Ctrl` + `Shift` + `J` или `F12`
- MacOS - `Command` + `Option` + `J`
